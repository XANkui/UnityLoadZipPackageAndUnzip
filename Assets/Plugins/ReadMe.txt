ICSharpCode.SharpZipLib.dll 的 使用说明

1、 在Unity中使用 ICSharpCode.SharpZipLib.dll时候，可能要与 I18N 系列文件一起使用，不然可能解压不成功，报错 ： System.NotSupportedException: Encoding 437 data could not be found. Make sure you have correct international codeset assembly installed and enabled.

2、这些文件也可以在 Unity 安装路径下的文件夹中找到 ：...\Unity\Editor\Data\Mono\lib\mono\unity
